# nuxt3-koa2-mongodb-starter

this is a template, including nuxt3 , koa2 , mongodb

utilizing a front-end and back-end separation approach.

front-end : **nuxt3**

back-end: **koa2 + mongodb**

### how to develop

front-end:

``` npm run dev:client ```

back-end: 

```npm run dev:server```

### how to deploy

front-end: 

```build:client```

back-end: 

```build:client```
